NES
===

An NES ROM made just to try out making NES Homebrew.

## Installing

### cc65
To build, you will need [cc65](https://github.com/cc65/cc65).
* `brew install cc65`

### FCEUX
To test, you will need [FCEUX](http://www.fceux.com/web/home.html).
* `brew install fceux`

## Building

To build:
* `make`

To test:
* `make test`

To clean:
* `make clean`
